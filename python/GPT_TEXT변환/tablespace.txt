tablespace TS_ICR_NORM_D001:테이블이나 인덱스가 저장될 테이블스페이스.
TS_ICR_NORM_D001이라는 이름의 테이블스페이스에 데이터를 저장. 
테이블스페이스는 데이터베이스 내에서 데이터 파일들을 그룹화하여 관리하는 구조.

PCTFREE 10: 테이블이나 인덱스의 각 블록에 대해 데이터를 저장할 때, 그 블록의 일정 비율을 여유 공간으로 남겨두는 값을 지정. 
새로운 데이터를 추가하거나 업데이트할 때, 해당 블록에 남길 여유 공간 비율입니다.
PCTFREE 10은 각 데이터 블록에서 10%를 여유 공간으로 남겨둔다고 설정하는 것. 
데이터 블록이 90%를 채운 후에는 더 이상 데이터를 추가하지 않고 새 블록을 사용.

INITRANS 2: 블록을 변경할 때, 최소한으로 필요한 트랜잭션 수를 지정. 데이터 블록을 여러 트랜잭션이 동시에 변경하려고 할 때 충돌을 피하기 위함.
INITRANS 2는 블록에 대해 최소한 2개의 트랜잭션이 동시에 작업할 수 있도록 설정.4

STORAGE ( MAXEXTENTS UNLIMITED ): 테이블이나 인덱스의 물리적인 저장 구조와 관련된 설정을 정의.
MAXEXTENTS UNLIMITED는 테이블이나 인덱스의 최대 확장(extent) 수를 무제한으로 설정. 
공간이 부족할 경우 추가 공간을 제한 없이 할당.

LOGGING: 데이터 변경에 대한 로그. 해당 테이블이나 인덱스에 대한 모든 변경 사항이 로그 파일에 기록.
